# Security Policy

Please report security issues privately to Armorer Labs:

dev@armorerlabs.com

Do not open public GitHub issues for vulnerabilities, bypasses, or sensitive
credential-handling bugs until Armorer Labs has had time to investigate and
coordinate a fix.

When reporting, include:

- affected version or commit
- reproduction steps
- expected and actual scanner output
- whether the issue affects credential redaction, semantic classification, or
  policy/tool-call handling
